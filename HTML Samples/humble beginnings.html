Humble Beginnings

Everyone has to start from somewhere and usually that 'somewhere' is a blank slate with big aspirations and very little knowledge on how to achieve them. Through persistance, dedication, a few doors of opportunity (whether self-made or pure luck) and massive amounts of caffiene, one can accomplish almost anything.

Granted, this page isn't very pretty, but to be fair, this is just a simple HTML page. As anyone learning Frontend Development can tell you, the 'pretty' stuff is within the styling of a webpage, and that uses a whole different language alltogether. You guessed it, I'm talking about CSS. But that is another topic for another time. For this page, I'm just going to use the bare bones, the 'starting point' of it all.

A lot of the styling that used to be done in simple HTML, is now done with CSS, but that doesn't mean that HTML is lacking in it's own right.
